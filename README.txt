# NURBS & B-Spline: De Boor & Knot Insertion

NURBS & B-Spline: De Boor & Knot Insertion: è un progetto universitario nell'ambito della computer grafica che permette, selezionati una serie di punti di controllo, di ottenere le NURBS e le B-Spline con il realtivo plot delle funzioni di base.

## Installation

Richiede l'istallazione di matlab R2020a o superiori.

## Usage

Per avviare correttamente l'applicazione:
1. Aprire il file user.m con il software matlab.
2. Selezionare nella Toolbar in alto la sezione EDITOR.
3. Premere il tasto Run.
4. Attendere il caricamento dell'interfaccia.
